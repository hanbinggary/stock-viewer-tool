#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build

# lint all JS/Vue component files in `src/`
npm run lint

```

#### 特点
````
1.接口使用的是腾讯股票接口

2.右键菜单可选择查看详情（雪球或股吧）

3.支持深沪港

4.支持置顶窗口

5.支持放置在屏幕顶端，能够自动缩起（类似QQ）

6.映射数字键盘，支持直接输入股票代码添加自选
````
