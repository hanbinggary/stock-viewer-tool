#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build

# lint all JS/Vue component files in `src/`
npm run lint

```

#### Features
````
1.接口使用的是腾讯股票接口

2.左键双击直接跳转查看详情，右键删除

3.支持深沪港

4.支持置顶窗口
````
